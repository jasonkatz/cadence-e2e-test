# Hello World Server

A minimal HTTP server built with Node.js's built-in `http` module.

## Getting Started

Start the server:

```sh
npm start
```

The server listens on port 3000 by default. To use a different port, set the `PORT` environment variable:

```sh
PORT=8080 npm start
```

## Expected Output

Once running, all requests return a `200` response with:

```
Hello, World!
```
